Anchors

Let's talk about anchors. Or the <a> element.

<a href="page location">link text</a> adds a link/hyperlink.

The href value is the path for the link.
This can be an absolute path like "http://www.google.ca". Or a relative (to the page) path.
You can also point to an anchor with "#id" to link to an ID named "id". This like will jump to an element or anchor with this name. If the anchor or element is in another page. put the page location first followed by the # and the ID name.
The anchor should be marked at the beginning of the content it is referring to.
Like this:
<a id="element">></a>
